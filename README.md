
# Infix to RPN Calc Engine

Source code to convert infix mathematics statements to Reverse Polish Notation.

Examples:

| Infix Statement                                 |     RPN format                    |   Result |
|-------------------------------------------------|-----------------------------------|----------|
| 3 + 4											                      | 3 4 +											        |	  7      |
| ( 3 + 4 )										                    | 3 4 +                             |          |
| ( ( 3 + 4 ) * 5 )								                | 3 4 + 5 *                         |          |
| ( ( 3 * 4 ) + 5 )								                | 3 4 * 5 +                         |          |
| ( 3 + 4 * 5 )									                  | 3 4 5 * +                         |          |
| 3 + 4 * 5										                    | 3 4 5 * +                         |          |
| 3 * 4 + 5										                    | 3 4 * 5 +                         |          |
| ( 3 * 4 + 5 )									                  | 3 4 * 5 +                         |          |
| 3 + 5 * 6										                    | 3 5 6 * +                         |          |
| 3 + ( 4 * 5 )	                   								| 3 4 5 * +                         |          |
| ( 3 / 5 ) + ( 2 * 4 )             							| 3 5 / 2 4											    |  8.6     |
| 3 * ( 4 + 5 )									                  | 3 4 5 + *                         |          |
| ( 3 + 4 ) * 5									                  | 3 4 + 5 *                         |          |
| ( 3 + 4 ) * ( 5 - 6 )							              | 3 4 + 5 6 - *                     |          |
| ( 3 + 4 ) * ( 5 - 6 ) / 7						            | 3 4 + 5 6 - * 7 /                 |          |
| ( ( 3 + ( 4 * 5 ) )								              | 3 4 5 * +                         |          |
| ( ( ( 3 + ( 4 * 5 ) ) )							            | 3 4 5 * +                         |          |
| ( 3 + 5 ) * ( 2 - 4 ) / 6						            | 3 5 + 2 4 - * 6 /                 |          |
| 3 / 4 + ( ( 5 - 4 ) / 6 * 7 )					          | 3 4 / 5 4 - 6 / 7 * +             |          |
| ( 5 * 2 ) - ( ( 3 + 4 * 7 ) + 8 / 6 ) * 9		    | 5 2 * 3 4 7 * + 8 6 / + 9 * -     |          |
| ( 5 * 2 ) - ( ( _3 + 4 * _7 ) + 8 / 6 ) * _9    | 5 2 * _3 4 _7 * + 8 6 / + _9 * -  |          |
